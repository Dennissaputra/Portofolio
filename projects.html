Projects

Computer Vision - Detection Sign language

This thesis project aims to develop a sign language detection system using YOLOv8, leveraging state-of-the-art computer vision techniques in machine learning and deep learning. The system features a responsive and user-friendly website interface designed to make the application accessible to the general public. A comprehensive dataset of sign language gestures was collected and processed to ensure accurate and representative detection results. For more details, the project is available on GitHub: Thesis Project.

Web Development Conference International

This project focuses on building a website to facilitate the online submission and review process for international conferences. The website features an intuitive, user-friendly design tailored for easy article submission and review, complete with integrated payment systems. Key functionalities include a submission form for international reviewers, a dynamic content management system for yearly updates, and full optimization for desktop, tablet, and mobile devices. Accessible via icoms.ummi.ac.id, the platform ensures a seamless and professional experience for all users.

Web Monitoring of Regent Election Volunteers

Developed a web-based system using Laravel, JavaScript, and DOM manipulation to manage and monitor volunteer activities during a regent election, ensuring transparency and efficiency. The system enables volunteers to input election data, providing real-time updates and accurate tracking of voting activities and trends across different regions. It features dynamic data visualization to display dominant voting preferences, aiding stakeholders in making informed decisions. This project highlights expertise in backend development, real-time data handling, and user-friendly front-end design for election systems.
